## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/DSA-Code.git
   ```

2. **Navigate to the desired folder**:
   ```bash
   cd DSA-Code/C++/DataStructures/Arrays
   ```

3. **Run the code**:
   - For C++:
     ```bash
     g++ filename.cpp -o output
     ./output
     ```
   - For Python:
     ```bash
     python filename.py
     ```
